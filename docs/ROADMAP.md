# Roadmap

План розвитку проєкту «Кролівництво від А до Я» у його сучасній версії.

## Найближчі кроки

- завершити міграцію контенту в нову структуру Next.js;
- покращити SEO для ключових сторінок;
- додати більше практичних розділів і калькуляторів;
- оптимізувати швидкість завантаження та UX.

## Середньострокові цілі

- розширити функціональність для користувача;
- покращити особистий кабінет і облік даних;
- додати AI-помічника або розумні рекомендації;
- посилити PWA/офлайн-можливості.

## Майбутні напрямки

- мультимовність (українська + англійська);
- інтеграція з калькуляторами годівлі та догляду;
- система нотаток, планів і сповіщень для власників кролів;
- спільнота, обговорення та зворотний зв’язок від користувачів.

## Принцип розвитку

Проєкт має рости не просто як сайт, а як корисний цифровий помічник для кролівників.
