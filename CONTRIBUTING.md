# Внесок у проєкт

Дякуємо, що хочете допомогти розвивати проєкт «Кролівництво від А до Я».

## Як допомогти

Ви можете:

- покращити контент і структуру сторінок;
- виправляти помилки або баги;
- покращувати UI/UX;
- доповнювати документацію;
- пропонувати нові функції.

## Перед початком

1. Форкніть репозиторій.
2. Створи гілку для своєї зміни.
3. Переконайся, що локально все працює.

## Рекомендації для змін

- дотримуйся існуючого стилю коду;
- не змінюй без потреби загальну структуру проєкту;
- додавай або оновлюй документацію разом із змінами;
- якщо змінюєш UI, перевіряй результат у браузері;
- якщо змінюєш логіку, перевіряй поведінку на реальному сценарії.

## Формат комітів

Пиши зрозумілі коміт-мереджі, наприклад:

- `feat: add new rabbit care page`
- `fix: correct metadata for breeding section`
- `docs: update contribution guide`

## Перевірка перед PR

Перед створенням pull request:

- запусти `npm run lint`;
- якщо є тести, запусти їх;
- переконайся, що збірка проходить через `npm run build`.

## Питання та пропозиції

Якщо у тебе є ідеї або питання, відкрий issue або напиши автору проєкту.
